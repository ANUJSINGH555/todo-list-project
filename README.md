# todo-list-project

THIS IS A SIMPLE ONLINE OR WEB BASED TODO LIST CREATED USING HTML, CSS, BOOTSTRAP AND JAVASCRIPT.
IT HAS FEATURES LIKE ADD A ITEM TO THE LIST OR DELETE AN ITEM FROM THE LIST AND CLEAR THE ENTIRE LIST
